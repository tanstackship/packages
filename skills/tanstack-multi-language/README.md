# TanStack Multi-Language Skill

> 为 AI 助手添加 TanStack Ship 多语言支持能力

这是一个 Claude Code/Cline/OpenClaw skill，帮助你快速为项目添加 i18n 多语言支持。

## 功能

- 🌍 自动生成多语言配置
- 📝 创建翻译文件模板
- 🔄 生成语言切换组件
- 🎯 支持 20+ 主流语言

## 安装

### Claude Code

```bash
cp -r skills/tanstack-multi-language ~/.claude/skills/
```

### Cline

```bash
cp -r skills/tanstack-multi-language ~/.cline/skills/
```

### OpenClaw

```bash
cp -r skills/tanstack-multi-language ~/.openclaw/skills/
```

## 使用

在 AI 助手中说：

> "使用 tanstack-multi-language skill 为我的项目添加中文和英文支持"

AI 助手会自动：
1. 分析你的项目结构
2. 生成 i18n 配置文件
3. 创建翻译文件模板
4. 添加语言切换组件
