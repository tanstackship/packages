# UTM Tracker WordPress Plugin

> Track and store UTM parameters in WordPress

## Installation

1. Download the plugin
2. Upload to `/wp-content/plugins/utm-tracker/`
3. Activate through the 'Plugins' menu in WordPress

## Usage

### Shortcodes

Display UTM source:
```
[utm_source default="direct"]
```
