## Installation

1. Download from WordPress plugin directory
2. Upload to `/wp-content/plugins/`
3. Activate via WordPress admin

## Features

- 📊 Automatic UTM capture
- 🍪 Cookie storage
- 📈 Analytics dashboard
- 🔗 Short links
- 📝 GDPR compliant

## Settings

1. Go to Settings > UTM Tracker
2. Configure storage preferences
3. Set consent requirements
4. Connect to analytics providers
